# Web Research Skill

## When to Use
- User asks you to find information about a topic
- You need current data that's not in your memory
- Verifying claims or checking facts
- Gathering sources for a report or summary

## How to Use
1. Use `web_search` with a specific, well-formed query
2. Review the search results — pick the 2-3 most relevant
3. Use `web_fetch` to get the full content of promising URLs
4. Extract the key facts, cite the sources
5. Store important findings via `store_memory` for future reference

## Guidelines
- Search queries should be specific — not "AI" but "AI agent frameworks comparison 2026"
- Always cite your sources by name
- Cross-reference claims across multiple sources
- Store the key takeaway in memory, not the raw page content
- If search results are thin, try rephrasing the query
- Prefer recent sources (last 12 months) for fast-moving topics
