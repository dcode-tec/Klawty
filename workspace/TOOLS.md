# Tools

| Tool | What it does |
|------|-------------|
| `file_read` | Read files from the workspace |
| `web_search` | Search the web |
| `web_fetch` | Fetch and parse a URL |
| `recall_memory` | Search your memory |
| `store_memory` | Save a fact to memory |
